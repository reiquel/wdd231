R&M ReStyle Closet

Site Plan Document

Site Name

R&M ReStyle Closet

This name represents a fashion marketplace where users can "restyle" their wardrobes by buying pre-loved clothing and shoes. The "R&M" adds a personal touch, representing the founders' initials, while "ReStyle Closet" clearly communicates the purpose of giving clothes a second life.

Site Purpose

The R&M ReStyle Closet website will:

  • Provide an online marketplace for buying and second-hand clothing and shoes
  • Offer advanced filtering options by brand, size, condition, and price range
  • Promote sustainable fashion practices and reduce textile waste
  • Create a community of fashion-conscious users who value affordability and sustainability
  • Include user profiles with ratings and reviews to build trust in the platform

Scenarios

Target audience questions that will drive content:

  • where can I purchase clothes quickly and safely?
  • Where can I find affordable, high-quality second-hand shoes?
  • What are the best ways to ensure I'm getting authentic items when buying second-hand?
  • How does buying second-hand clothing help my finances?

Color Schema

The color palette was chosen to reflect sophistication (deep purple), creativity (soft purple), and approachability (dusty pink):

Primary
#5E548E
Secondary
#9F86C0
Accent
#E0B1CB
Background
#F8F4F9

Color Applications:

  • Primary (#5E548E): Navigation bar, primary buttons, main headings
  • Secondary (#9F86C0): Secondary buttons, borders, accents
  • Accent (#E0B1CB): Highlights, hover states, decorative elements
  • Background (#F8F4F9): Page background for clean, soft appearance

Typography

The typography combines elegant serif headings with clean, readable sans-serif body text:

Playfair Display (Heading Font)

Used for all headings (H1, H2, H3) to create an elegant, fashion-forward impression.

Montserrat (Body Font)

Used for all body text, navigation, buttons, and other interface elements for optimal readability across devices.
